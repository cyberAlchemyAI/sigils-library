# Architecture Inventory

This folder stores structured inventory data and schemas for the architecture package.

## Files

- `architecture-inventory.schema.json` defines the portable inventory shape.
- Repository-specific inventory files may be added here when useful.

## Inventory Contents

The inventory should capture:

- layers,
- modules,
- concepts,
- relationships,
- dependency rules,
- testing obligations,
- observability obligations,
- gaps and unknowns.

Keep large evidence details in markdown cards and store compact references in structured inventory files.
