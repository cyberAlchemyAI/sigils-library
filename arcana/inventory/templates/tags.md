# Inventory Tags

This file defines reusable tags for generated inventory pages.

## Tag Rules

- Prefer stable, lowercase, hyphenated tags.
- Add a tag here before using it repeatedly.
- Avoid near-duplicates; add aliases when needed.
- Tags describe content, lifecycle, confidence, and source type.
